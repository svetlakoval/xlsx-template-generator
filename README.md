# XLSX Template Generator

Веб-приложение для генерации тестовых `.xlsx`-файлов по загруженному Excel-шаблону.

## Возможности

- Загружает `.xlsx` и анализирует выбранный лист.
- Берёт названия колонок из указанной строки заголовков.
- Использует строки ниже заголовков только для распознавания структуры и примеров генерации.
- Генерирует новый файл без исходных строк-примеров.
- Позволяет указать итоговое количество строк данных.
- Позволяет игнорировать служебные строки при анализе.
- Автоматически предлагает типы колонок: текст, число, дата, email, телефон, ФИО, фамилия, имя, отчество, пол, компания, город, гражданство/страна, адрес, номер бланка, кем выдан, UUID, boolean, список, формула.
- Сохраняет форматирование заголовков, колонок и строки-образца.

## Особенности генерации

- Даты генерируются в формате `дд.мм.гггг`.
- Телефоны генерируются без `+`, например `79778343974`.
- Фамилия, имя, отчество и пол согласуются в рамках одной строки.
- Адреса и номера бланков продолжаются по числовому шаблону из примеров.
- Поле «Кем выдан» генерирует значения вида `ОВД ...`, `УФМС ...`, `ГУ МВД ...`.

## Запуск

```bash
npm install
npm run dev
```

Открой адрес, который покажет Vite, обычно `http://localhost:5173`.

## Сборка

```bash
npm run build
```
